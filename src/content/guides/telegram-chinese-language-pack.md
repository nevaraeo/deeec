## 原理：什么是“Telegram 中文版”

Telegram（纸飞机）官方不提供单独的中文安装包。**Telegram 中文版 = 官方原版 + 官方简体中文语言包**。语言包通过一个 `t.me` 官方深链一键应用，安全、可逆。

## 方法一：一键深链（推荐）

1. 先安装并登录 Telegram 官方原版（[下载入口](/downloads/)）；
2. 在浏览器打开本站每页底部的「应用简体中文语言包」按钮（官方深链 `t.me/setlanguage/...`）；
3. 系统唤起 Telegram → 弹出语言切换提示 → 点击 **Apply / 应用**；
4. 界面立即变为简体中文。

## 方法二：客户端内手动切换

打开 Telegram → **Settings（设置）** → **Language（语言）** → 选择 **简体中文** → 确认。

## 各平台差异提示

| 平台 | 打开深链的方式 |
| --- | --- |
| Windows / Mac / Linux | 系统默认浏览器打开深链，会自动唤起桌面客户端 |
| Android | 用手机浏览器打开深链，唤起 Telegram App |
| iOS | 用 **Safari** 打开深链效果最佳 |

## 安全提醒

不要为了“中文”去下载第三方汉化包 / 破解版，详见 [Telegram 下载安全指南](/blog/telegram-download-safety/)。相关：[纸飞机中文版](/paper-airplane-chinese/)、[Telegram 中文说明](/telegram-chinese/)。
