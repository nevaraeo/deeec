装好 Telegram 打开是英文，是每个新用户的“第一道坎”。其实把它变成简体中文非常简单、安全，而且随时能改回去。这篇教程适用于**所有平台**（Windows、Mac、Linux、安卓、iPhone），照着做一分钟搞定，并解答“为什么没反应”这类常见问题。

## 先理解原理：所谓“Telegram 中文版”是什么

这一点想通了，你就不会再去网上找高风险的“汉化版”。Telegram 官方**不提供单独的中文安装包**，我们说的“Telegram 中文版”其实是：

> 官方原版 App ＋ 官方简体中文语言包 ＝ 完整简体中文界面

语言包通过一个 `t.me` 官方深链一键应用，**只改显示文字、不动客户端本体**，因此安全、可逆，对账号和聊天毫无影响。

## 方法一：一键深链（推荐，最快）

1. 先安装并登录 Telegram 官方原版（还没装从 [下载入口](/downloads/) 获取）；
2. 在浏览器里打开本站**每个页面底部的「应用简体中文语言包」按钮**（它就是官方深链 `t.me/setlanguage/...`）；
3. 系统会自动唤起 Telegram，并弹出语言切换提示；
4. 点击 **Apply / 应用**——界面瞬间变成简体中文，菜单、设置、按钮全部中文化。

## 方法二：在客户端里手动切换（备用）

不想用深链也行，直接在 App 内选：

打开 Telegram → **Settings（设置）** → **Language（语言）** → 在列表里选 **简体中文（Simplified Chinese）** → 确认。

部分版本的语言列表需要联网加载，稍等片刻即可看到选项。

## 不同平台，打开深链的小差异

| 平台 | 打开深链的方式 |
| --- | --- |
| Windows / Mac / Linux | 用系统默认浏览器打开深链，会自动唤起桌面客户端 |
| Android | 用手机浏览器打开深链，唤起 Telegram App 后确认 |
| iOS | 建议用 **Safari** 打开深链，唤起 App 的成功率最高 |

## 没反应？这样排查

- 先确认你点的是「**Apply / 应用**」，而不是“Cancel / 取消”；
- 如果还是英文，**完全退出 Telegram 再重新打开**一次；
- 仍不行就用方法二，在 Settings → Language 里手动选简体中文；
- 多设备通常会随账号同步语言；个别旧版本需要在每台设备上各确认一次。

想换回英文？随时回到 Settings → Language 切换即可，操作可逆、不丢任何数据。

## 一句话安全提醒

请只用**官方语言包**，不要为了中文去装第三方“汉化版/破解版”——那类文件可能植入广告或窃取验证码（详见 [Telegram 下载安全指南](/blog/telegram-download-safety/)）。
