在电脑上用 Telegram，体验比手机舒服得多——大屏打字快、传文件直接拖拽、还能多账号挂着。而且 Telegram 电脑版是**独立客户端**：登录一次之后，哪怕手机关机，电脑端照样收发消息。这篇就手把手带你从下载到改中文走完整个流程，整个过程大约五分钟，新手也能一次成功。

## 开始前，先准备两样东西

- 一台 **64 位的 Windows 7 / 8 / 10 / 11** 电脑（绝大多数电脑都满足）；
- 一个能接收验证码的**手机号**（已有 Telegram 账号或准备新注册都行）。

## 步骤一：从官方源下载安装包

进入本站 [Telegram 电脑版下载页](/windows/)，点击「下载 Windows 版」。下载地址**直连 telegram.org 官方服务器**，得到一个名为 `tsetup.x.x.x.exe` 的安装程序。

> 切记只从官方源下载，不要去搜“Telegram 中文版.exe / 汉化破解版”——那类文件常被二次打包甚至盗号（原因见 [下载安全指南](/blog/telegram-download-safety/)）。中文界面我们在第四步用官方语言包来实现，既安全又完整。

## 步骤二：运行安装

1. 双击下载好的安装包；
2. 选择安装界面语言（先用 English 即可，后面会改中文）；
3. 选择安装目录，**保持默认就行**；
4. 一路点「下一步 / Next」完成安装，结束时勾选「运行 Telegram」。

如果是便携版（Portable），解压后直接运行其中的 Telegram.exe 即可，无需安装。

## 步骤三：用手机号登录

1. 打开 Telegram，点击 **Start Messaging**；
2. 选择国家区号（中国大陆是 +86），输入手机号；
3. 输入收到的验证码——它可能是短信，也可能发到你已登录的手机 Telegram 里；
4. 登录成功，**已有账号会自动从云端同步回全部聊天记录**。

从这一步起，电脑端就能独立工作了，手机不在身边也不影响。

## 步骤四：把界面切换成简体中文

1. 在浏览器里打开本站任意页面底部的「**应用简体中文语言包**」按钮（这是官方深链）；
2. 系统会自动唤起 Telegram，并弹出语言切换提示；
3. 点击 **Apply / 应用**，界面立刻变成简体中文。

这一步只改显示语言、安全可逆，日后想换回英文，在「设置 → Language」里随时切换。

## 装好之后，建议再做一件事

花一分钟开启**两步验证**（设置 → 隐私与安全 → 两步验证），给账号加一道密码锁，能有效防盗号（详见 [安全设置指南](/blog/telegram-liangbu-yanzheng-anquan/)）。
