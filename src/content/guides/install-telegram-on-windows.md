## 准备

- 一台 64 位 Windows 7 / 8 / 10 / 11 电脑
- 一个可接收验证码的手机号（已注册或新注册 Telegram）

## 步骤一：下载官方安装包

进入本站 [Telegram 电脑版下载页](/windows/)，点击「下载 Windows 版」。下载地址直连 `telegram.org` 官方服务器，得到 `tsetup.x.x.x.exe`。

## 步骤二：运行安装

双击安装包 → 选择界面语言（English）→ 选择安装目录（默认即可）→ 一路「下一步」完成安装。安装结束勾选「运行 Telegram」。

## 步骤三：登录账号

1. 打开 Telegram，点击 **Start Messaging**；
2. 选择国家区号，输入手机号；
3. 输入手机收到的验证码（已有账号会自动云端同步全部聊天）。

## 步骤四：切换简体中文

1. 在浏览器打开本站任意页面底部的「应用简体中文语言包」按钮（官方深链）；
2. 系统唤起 Telegram，弹出语言提示，点击 **Apply / 应用**；
3. 界面立即变为简体中文。

完成。更多见 [中文设置专项教程](/guides/telegram-chinese-language-pack/) 与 [Windows 11 专版](/telegram-download-for-windows-11/)。
